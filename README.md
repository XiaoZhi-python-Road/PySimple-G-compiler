## 系统平台要求：
-----------------------------------------------------------------------------------------------------------------------
### 1.仅限于Windows 平台，仅支持Windows 10 22H2、Windows 11 24 H2、Windows 11 25H2 以及 2026年秋季的Windows 11 26H2
### 2.仅限于原版系统，确保UAC挡位为默认的挡位！
### 3.对于第三方盗版的系统将不再支持......
-----------------------------------------------------------------------------------------------------------------------
### 使用方式如下：
-----------------------------------------------------------------------------------------------------------------------
1.请先下载好G++ Tools 文件夹和编译文件夹，且放在一个干净的磁盘中，比如 E:\kun  
2.进入编译文件夹，使用解压缩软件解压MinGW，注意：是解压到当前文件夹，确保MinGW_x64文件夹在编译文件夹中跟的目录下！    
3.进入G++ Tools文件夹，双击运行 PySimple G++ Tools.exe，等待启动，将要转换pyd的py文件放在G++ Tools\Scripts的目录中，这样工具就能监听到Scripts目录中有py文件！    
4.点击准备就绪后，等待生成完毕，生成的setup.py会在G++ Tools文件夹下跟的目录下！将setup.py剪切粘贴到编译文件夹中！
